## Previewing the site

This site is powered by [Zola](https://www.getzola.org/).

Install on MacOS:

```sh
brew install zola
```

Install on Linux (get `url` from the [Releases page](https://github.com/getzola/zola/releases)):

```sh
curl -L [url] | tar -xz && sudo mv zola /usr/local/bin/
```

Run:

```sh
zola serve
```
